CSS: A Website's Skin

CSS, which stands for Cascading Style Sheets, is a style sheet language that allows you to style the layout of a website. If this page had no CSS elements in it, there wouldn't be a sidebar, and the background would just be a bland white. Without CSS, fonts and other style choices would not be available.
